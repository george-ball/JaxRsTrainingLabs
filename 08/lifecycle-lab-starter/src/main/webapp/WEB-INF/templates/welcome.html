Welcome to the Lifecycle Lab

This HTML file lives at /WEB-INF/templates/welcome.html inside the WAR. It was read using ServletContext.getResourceAsStream().

This demonstrates that you can serve server-side content from files bundled inside the WAR without exposing them directly via the web server.

Key point: Files inside /WEB-INF/ cannot be accessed directly by a browser URL — they must be served programmatically.
